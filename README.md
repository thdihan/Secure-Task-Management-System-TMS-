### Steps to run the application:

-   Requirement
    -   MongoDB in local pc
    -   npm
-   Install npm package in both _"frontend"_ and _"backend"_ folder using `npm install` command.
-   Run frontend using `npm run dev`
-   Run backend using `npm start`
